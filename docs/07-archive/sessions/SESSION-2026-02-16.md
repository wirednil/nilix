### v0.14.0 - Dynamic Select con `<in-table>`

**Sintaxis nueva:**
```xml
<!-- Select dinámico desde BD -->
<field id="clieno" label="Cliente" type="select" size="6">
    <in-table table="clientes" key="clieno" display="nombre">
        <copy from="nombre" to="nombre"/>
        <copy from="direc" to="direc"/>
    </in-table>
</field>

<!-- Select estático -->
<field id="tipo_iva" label="Tipo IVA" type="select" size="2">
    <options>
        <option value="RI">Responsable Inscripto</option>
        <option value="CF">Consumidor Final</option>
    </options>
</field>
```

**Lógica:**
| type | Sub-elemento | Resultado |
|------|--------------|-----------|
| select + options | - | Select estático |
| select + in-table | - | Select dinámico (carga al focus) |
| text/number + in-table en validation | - | Validación blur + copy |

**Modificados:**
- `js/components/FormRenderer.js` - Detecta in-table en selects, lookupConfig
- `js/components/fieldRenderer/InputField.js` - type="dynamic-select", attachDynamicSelectHandlers()

**Formularios:**
- `forms/clie.xml` - Ejemplo completo con ambas sintaxis
- Eliminado `forms/test-dynamic-select.xml`
