## Lecciones Aprendidas

1. **CDN jsDelivr con `+esm` no funciona bien con DuckDB-WASM** - Usar `esm.sh` con deps explícitas
2. **Worker creation requiere blob URL** - `importScripts()` desde CDN
3. **WebAssembly.suspend detection** - Usar `WebAssembly.suspend` (no `_suspend`)
4. **Version matters** - v1.33.0 no existe, usar v1.29.0
5. **Fallback es crítico** - JS loops como backup si DuckDB falla
