## Lo Implementado

### v0.14.3 - The Monospace Web Integration

**Inspiración:** https://owickstrom.github.io/the-monospace-web

**Cambios principales:**

| Aspecto | Antes | Después |
|---------|-------|---------|
| Fuente | Separadas (`--label-font`, `--input-font`) | Unificada (`--font-family`) |
| Input height | Auto | `calc(var(--line-height) * 2)` |
| Input padding | `0.4rem` | `calc(1ch - var(--border-thickness))` |
| Label-input gap | `0.5rem` | `0` |
| Font inheritance | Explícita | `font: inherit` |
| Dark mode shadow | Verde `#00ff00` | Blanco `#ffffff` |

**Archivos modificados:**
- `css/styles.css` - Monospace patterns, input sizing, responsive
- `index.html` - CRT toggle button
- `js/main.js` - CRT toggle logic

**Variables CSS nuevas:**
```css
--font-family: ui-monospace, 'Cascadia Code', 'JetBrains Mono', ...;
--line-height: 1.2rem;
--border-thickness: 2px;
font-optical-sizing: auto;
font-variant-numeric: tabular-nums lining-nums;
```

**Responsive border-box móvil:**
```css
@media (max-width: 650px) {
    .border-box .horizontal-container {
        flex-direction: column;
    }
    .border-box input { width: 100% !important; }
}
```
